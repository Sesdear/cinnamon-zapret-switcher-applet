# cinnamon-zapret-switcher-applet
Applet for cinnamon to switch systemctl zapret status
## Install guide
1. Clone Repo: `git clone https://github.com/Sesdear/cinnamon-zapret-switcher-applet.git && cd cinnamon-zapret-switcher-applet`
2. Run install script: `chmod +x install.sh && ./install.sh`
   or install manual: `cp -r zapretswitcher-cinnamon@sesdear.github.io ~/.local/share/cinnamon/applets/`
